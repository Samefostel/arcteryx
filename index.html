Про нас

Gorp Core - це бренд, заснований на спільноті, який надає високоякісне обладнання для активного відпочинку, розроблене для комфорту та довговічності. Ми віримо в те, щоб зробити кожну пригоду незабутньою, забезпечуючи мандрівників правильними інструментами.

Наше обладнання

Окуляри

Міцні, водонепроникні рюкзаки для будь-якого рельєфу.

Одяг

Досліджуйте з комфортом з нашими міцними та універсальними черевиками для походів.

Кросівки

Стильний та функціональний одяг, розроблений для активного відпочинку.
